Map Path is currently off. Click below to enable it.

In the Extensions tab, check Map Path, then when prompted choose Always Allow on Every Website.

New to Safari extensions?

  1. Open Safari.
  2. Choose Safari › Settings… (⌘,), then click Extensions.
  3. Check the box next to Map Path.
  4. When prompted about website access, choose Always Allow on Every Website.

Turn it on — iPhone & iPad (iOS / iPadOS 26.5)

  1. Open Settings.
  2. Tap Apps, then Safari, then Extensions.
  3. Tap Map Path and turn the toggle on.
  4. Under Permissions, tap All Websites and choose Allow.

Or, from any Safari tab: tap the “ᴀA” button in the address bar → Manage Extensions → toggle Map Path on.

Map Path rewrites map links from Google Maps, Waze, Bing Maps, HERE WeGo, and geo: URIs so clicks open in Apple Maps. Everything runs on your device — granting access to all websites is what lets Map Path see map links, nothing else leaves Safari.
