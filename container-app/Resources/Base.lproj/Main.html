Takes you to Map Path's iOS settings. Turn on Allow Extension, then under Permissions tap All Websites and choose Allow — not Ask.

Step by step

  1. Open Settings → Apps → Safari → Extensions. (Older iOS: Settings → Safari → Extensions.)
  2. Tap Map Path, then turn Allow Extension on.
  3. Under Permissions, tap All Websites and choose Allow.

This step matters. All Websites must be Allow — if it's left on Ask, Map Path can't rewrite map links and clicks will still open Google or Waze.

Optional: turn on Allow in Private Browsing if you use Private tabs and want Map Path to work there too.

Map Path is installed

Enable it in Safari whenever you're ready — your map links will start opening in Apple Maps from there.

Map Path rewrites map links from Google Maps, Waze, Bing Maps, HERE WeGo, and geo: URIs so clicks open in Apple Maps. Everything runs on your device — granting access to all websites is what lets Map Path see map links, nothing else leaves Safari.
